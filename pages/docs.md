SquishMark is a blogging engine that reads your Markdown straight from a GitHub repo and renders it on request. Push a post and the site updates in seconds.

New here? Start with [Getting started](/docs/getting-started) to go from an empty repo to a live blog. After that:

- [Configuration](/docs/configuration) covers `config.yml` and the environment variables SquishMark reads.
- [Theming](/docs/theming) walks through the bundled themes and how to build your own with Jinja2.
- [Frontmatter](/docs/frontmatter) lists every field you can set at the top of a post or page.
