# Welcome to Your New Blog

This is your first post. Edit or delete it to get started.

## Writing Posts

Posts are written in Markdown with YAML frontmatter. Save them in the `posts/` directory with a filename like `YYYY-MM-DD-slug.md`.

## Code Highlighting

SquishMark uses Pygments for syntax highlighting. Here's an example:

```python
def hello_world():
    print("Hello from SquishMark!")

if __name__ == "__main__":
    hello_world()
```

And here's some JavaScript:

```javascript
const greeting = (name) => {
  console.log(`Hello, ${name}!`);
};

greeting("World");
```

## What's Next?

- Edit `config.yml` to set your site title and author
- Add more posts to the `posts/` directory
- Create static pages in the `pages/` directory
- Customize your theme in the `theme/` directory
