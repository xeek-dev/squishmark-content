This is the official site for SquishMark, and it runs on SquishMark itself. Every post and page here is a Markdown file in a public GitHub repo, fetched and rendered at request time by the engine.

The site is just getting started. Coming soon:

- Documentation: getting started, configuration, and theming guides
- A changelog page tracking releases
- Release posts explaining what changed and why

In the meantime, the [SquishMark repository](https://github.com/xeek-dev/squishmark) has a README that covers the basics, and the [squishmark-starter](https://github.com/xeek-dev/squishmark-starter) template is the fastest way to try it out.
